# Sntledge blog

A single, minimal blog page. `index.html` is a self-contained article —
no build step, no external dependencies.

Current post: **How to Stop Revenge Trading** (trading psychology).

## Run locally

Open `index.html`, or serve the folder:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Deploy

Push to any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages);
the repo root is the site root.
